# Test Cases

## Login Functionality

### Test Case 1: Successful login
- Steps:
  1. Open login page
  2. Enter valid username and password
  3. Click login
- Expected result:
  User is logged in successfully

### Test Case 2: Invalid password
- Steps:
  1. Enter valid username
  2. Enter wrong password
  3. Click login
- Expected result:
  Error message is displayed
